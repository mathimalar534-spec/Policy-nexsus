# Vulnerability Management Policy

**Version:** v2.1  
**Owner:** Priya Nair (Operations)  
**Last Reviewed:** 2025-11-17  
**Status:** Retired

## 1. Purpose

This document establishes the vulnerability management policy for the organization. It is maintained by the Operations department and applies to all personnel and systems in scope below.

## 2. Scope

This policy applies to the following groups: admins, all users, contractors, employees.

## 3. Obligations

### 3.1 Change

Change management obligations govern how modifications to production systems are approved.

- **[Recommended]** All Users should change.

### 3.2 Endpoint

Endpoint obligations apply to laptops, workstations, and other managed endpoints.

- **[Mandatory]** Contractors shall endpoint.
- **[Prohibited]** Contractors prohibited endpoint.

### 3.3 Password

Password obligations govern how credentials are created, stored, and rotated.

- **[Recommended]** Admins recommended password.
- **[Recommended]** Admins should password.

### 3.4 Api

API obligations govern how internal and external interfaces must be secured.

- **[Mandatory]** Contractors shall api.

### 3.5 Cloud

Cloud obligations apply to infrastructure and services hosted with cloud providers.

- **[Prohibited]** Admins prohibited cloud. (Reference: DES)

### 3.6 Asset

Asset management obligations cover the tracking and disposal of company assets.

- **[Prohibited]** Admins prohibited asset.

### 3.7 Mobile

Mobile device obligations apply to any device used to access company resources.

- **[Mandatory]** Employees shall mobile.

### 3.8 Backup

Backup and recovery obligations ensure business continuity in the event of data loss.

- **[Recommended]** All Users should backup. (Reference: GDPR 2016)

### 3.9 Monitoring

Monitoring obligations define what systems must be observed and alerted on.

- **[Prohibited]** Contractors prohibited monitoring.

## 4. References

This policy's technical baseline is defined with reference to WEP wireless encryption, as adopted at the time of the last major revision.
