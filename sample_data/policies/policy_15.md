# Backup & Recovery Policy

**Version:** v2.4  
**Owner:** Noah Patel (Compliance)  
**Last Reviewed:** 2025-04-01  
**Status:** Draft

## 1. Purpose

This document establishes the backup & recovery policy for the organization. It is maintained by the Compliance department and applies to all personnel and systems in scope below.

## 2. Scope

This policy applies to the following groups: admins, all users, contractors, developers, employees.

## 3. Obligations

### 3.1 Provisioning

Provisioning obligations define how user accounts are created and deprovisioned.

- **[Recommended]** Contractors recommended provisioning.

### 3.2 Endpoint

Endpoint obligations apply to laptops, workstations, and other managed endpoints.

- **[Prohibited]** All Users prohibited endpoint.

### 3.3 Change

Change management obligations govern how modifications to production systems are approved.

- **[Prohibited]** Developers prohibited change.
- **[Prohibited]** Employees prohibited change.

### 3.4 Password

Password obligations govern how credentials are created, stored, and rotated.

- **[Recommended]** Admins should password. (Reference: SOX 2002)

### 3.5 Logging

Logging obligations define what activity must be recorded and for how long.

- **[Prohibited]** Employees prohibited logging.

### 3.6 Privacy

Privacy obligations govern the collection, use, and disclosure of personal data.

- **[Recommended]** Developers recommended privacy.

### 3.7 Network

Network obligations govern segmentation, firewalling, and perimeter defenses.

- **[Recommended]** Developers recommended network.
- **[Mandatory]** Developers must network.

### 3.8 Data retention

Data retention obligations set minimum and maximum retention periods for records.

- **[Mandatory]** Developers must data retention.

### 3.9 HR

HR obligations apply to onboarding, offboarding, and personnel security practices.

- **[Recommended]** Contractors recommended HR.

### 3.10 Physical

Physical security obligations cover access to offices, data centers, and equipment.

- **[Mandatory]** Admins shall physical.
