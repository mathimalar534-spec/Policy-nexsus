# Patch Management Policy

**Version:** v2.3  
**Owner:** Ava Johnson (Compliance)  
**Last Reviewed:** 2025-12-24  
**Status:** Draft

## 1. Purpose

This document establishes the patch management policy for the organization. It is maintained by the Compliance department and applies to all personnel and systems in scope below.

## 2. Scope

This policy applies to the following groups: admins, all users, contractors, developers, employees.

## 3. Obligations

### 3.1 Cloud

Cloud obligations apply to infrastructure and services hosted with cloud providers.

- **[Mandatory]** Employees shall cloud.

### 3.2 Monitoring

Monitoring obligations define what systems must be observed and alerted on.

- **[Mandatory]** Admins shall monitoring.

### 3.3 Backup

Backup and recovery obligations ensure business continuity in the event of data loss.

- **[Recommended]** All Users recommended backup.

### 3.4 Network

Network obligations govern segmentation, firewalling, and perimeter defenses.

- **[Recommended]** Employees should network.

### 3.5 Physical

Physical security obligations cover access to offices, data centers, and equipment.

- **[Recommended]** All Users should physical.
- **[Recommended]** Employees should physical.

### 3.6 Provisioning

Provisioning obligations define how user accounts are created and deprovisioned.

- **[Mandatory]** All Users must provisioning.

### 3.7 Asset

Asset management obligations cover the tracking and disposal of company assets.

- **[Mandatory]** Employees must asset.
- **[Mandatory]** Contractors required asset. (Reference: NIST SP 800-53 Rev 4)

### 3.8 Vendor

Vendor obligations apply to the selection and oversight of external suppliers.

- **[Mandatory]** Contractors must vendor.

### 3.9 Patch

Patch management obligations set expectations for applying security updates.

- **[Recommended]** Developers should patch.

## 4. References

This policy's technical baseline is defined with reference to the Safe Harbor framework (invalidated in 2015), as adopted at the time of the last major revision.
