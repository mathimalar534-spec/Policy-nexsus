# Third-Party Risk Policy

**Version:** v1.4  
**Owner:** Fatima Noor (HR)  
**Last Reviewed:** 2024-11-06  
**Status:** Retired

## 1. Purpose

This document establishes the third-party risk policy for the organization. It is maintained by the HR department and applies to all personnel and systems in scope below.

## 2. Scope

This policy applies to the following groups: admins, all users, contractors, developers, employees.

## 3. Obligations

### 3.1 Network

Network obligations govern segmentation, firewalling, and perimeter defenses.

- **[Recommended]** Employees recommended network.

### 3.2 Endpoint

Endpoint obligations apply to laptops, workstations, and other managed endpoints.

- **[Mandatory]** Employees shall endpoint.

### 3.3 Cloud

Cloud obligations apply to infrastructure and services hosted with cloud providers.

- **[Mandatory]** Developers required cloud.
- **[Recommended]** Admins recommended cloud.
- **[Prohibited]** Developers prohibited cloud. (Reference: FTP)

### 3.4 Patch

Patch management obligations set expectations for applying security updates.

- **[Mandatory]** Contractors shall patch.

### 3.5 Password

Password obligations govern how credentials are created, stored, and rotated.

- **[Mandatory]** Admins shall password.

### 3.6 Data retention

Data retention obligations set minimum and maximum retention periods for records.

- **[Mandatory]** Contractors required data retention.

### 3.7 Mobile

Mobile device obligations apply to any device used to access company resources.

- **[Mandatory]** All Users required mobile. (Reference: DES)

### 3.8 Encryption

Encryption obligations specify how data must be protected at rest and in transit.

- **[Mandatory]** Employees required encryption.

### 3.9 Access

Access control obligations define who may reach which systems and data.

- **[Recommended]** Admins recommended access. (Reference: SOX 2002)
- **[Prohibited]** All Users prohibited access. (Reference: TLS 1.0)

## 4. References

This policy's technical baseline is defined with reference to the pre-2018 draft of the EU Data Protection Directive (superseded by GDPR), as adopted at the time of the last major revision.
